# Epic 01: 基础聊天界面

## Epic 信息
- **Epic 编号**: Epic-01
- **Epic 名称**: 基础聊天界面
- **创建日期**: 2025-12-29
- **优先级**: High

## 业务目标

DouDouChat 作为一个开源的 AI 角色扮演聊天应用，需要首先建立一个稳定、美观的前端基础。本 Epic 的目标是实现第一个可用版本的核心前端界面。

这个版本聚焦于"最小可行产品"，让用户能够体验基本的 AI 对话功能。通过实现聊天界面、会话管理、人格设置和 LLM 配置四个核心模块，为后续的长期记忆、多模态交互等高级功能奠定基础。

本 Epic 仅涉及前端开发，使用 Mock 数据模拟后端响应，确保 UI/UX 的完整性和可测试性。

## 范围描述

### 包含内容
- **聊天界面**：消息展示、输入发送、实时响应展示
- **会话管理**：会话列表、新建/删除/切换会话
- **人格设置**：预设人格选择、基础人格参数配置
- **LLM 配置**：API 密钥配置、模型选择、基础参数调整
- **响应式布局**：PC 端优先，支持移动端适配

### 不包含内容
- ❌ 后端 API 开发（使用 Mock 数据）
- ❌ 自定义角色创建功能
- ❌ 长期记忆功能
- ❌ TTS/STT 多模态交互
- ❌ 用户登录/注册系统
- ❌ 数据持久化到服务端

## 用户旅程

```
用户打开应用 
    → 看到会话列表（左侧边栏）
    → 新建或选择一个会话
    → 进入聊天界面
    → 输入消息并发送
    → 看到 AI 实时响应
    → 可随时切换会话或调整设置
```

## 技术约束

- **前端框架**：Vue 3 + Vite
- **UI 组件库**：ai-elements-vue（基于 shadcn-vue）
- **样式**：Tailwind CSS
- **数据模拟**：前端 Mock 数据，模拟 AI 响应
- **状态管理**：Vue Composition API / Pinia
